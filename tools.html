Tools and Code

Here I make openly available the tools and code that I have developed a number of different analysis methods. Most of the tools and code is written in the statistical software R.

WildlifeDI

A suite of tools (in R) for measuring dynamic interaction in wildlife telemetry data.

stampr

Methods for for performing spatial-temporal analysis of polygon movement.

Time Geographic home range analysis

A set of methods for performing time geographic home range anaysis.
