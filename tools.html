Tools and Code

Here I make openly available the tools and code that I have developed a number of different analysis methods. Most of the tools and code is written in the statistical software R.


wildlifeDI

A suite of tools or measuring dynamic interaction in wildlife telemetry data in R.


wildlifeTG

A set of methods for time geographic analysis of wildlife telemetry data in R.


stampr

Methods for for performing spatial-temporal analysis of moving polygons in R.


PathInterpolatR

Code for for performing path interpolation in R.
